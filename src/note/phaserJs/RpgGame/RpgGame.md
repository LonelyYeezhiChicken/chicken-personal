### 關於 Phaser 3

- 什麼是 Phaserjs 呢?
  1. 他是一個 Game Framework
- 為什麼要選擇它呢?
  1. 首先因為他的文件非常完整，而且一直持續在更新
  2. 接下來最重要的是他還支持 TypeScript
  3. 因為都屬於靜態資源，因此可以使用 Github page 建立屬於你的小遊戲
