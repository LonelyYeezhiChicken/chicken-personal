# 📊 高階統計圖表分析

本章節只做一件事：說明控制圖以外的三種圖表**各回答什麼問題**。軌道 A 選讀。

## 讀完本篇你能回答

- 直方圖、盒鬚圖、概率圖各適合什麼場景？
- 三種圖如何與控制圖交叉驗證？

## 1. 三種圖各管什麼

| 圖表 | 回答的問題 |
|------|-----------|
| **直方圖** + 常態擬合 | 分佈有沒有能力？中心偏沒偏？ |
| **盒鬚圖** | 哪台機台/腔體明顯較差？ |
| **概率圖** | 常態假設是否成立？ |

## 2. 建議搭配方式

1. 控制圖 → 製程**穩不穩**
2. 直方圖 → 製程**夠不夠窄**
3. 盒鬚圖 → 機台間**均不均**

非常態時概率圖偏離對角線，應改用 [`advanced-calculation`](../engine/advanced-calculation.md) 的分位數界限。
