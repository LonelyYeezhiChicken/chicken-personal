# 📊 高階統計圖表分析

本章節介紹 SPC 系統中除趨勢圖外的進階視覺化工具。這些圖表提供了對製程健康度的多維度審視。

## 1. 直方圖與常態擬合

直方圖是評估 $C_{pk}$ 最直觀的工具。
- **常態擬合曲線**：自動繪製常態分佈曲線。
- **規格視覺化**：疊加 $USL, Target, LSL$。工程師可一眼看出分佈中心是否偏離。

## 2. 盒鬚圖與機台對比分析 (Boxplots)

盒鬚圖是用於「機台差異分析」的最佳工具。
- **視覺組成**：中位數、IQR、離群點。
- **平行對比**：橫向排列不同機台品質數據，鎖定表現較差的腔體。

## 3. 概率分佈圖 (Probability Plot)

用於嚴謹評估數據是否符合常態分佈：
- **判讀依據**：若點位落在對角線，代表常態分佈假設成立。

## 4. 領域專家思維：圖表的綜合運用

專家利用多種工具進行交叉驗證：
1.  **趨勢圖**：確認製程是否「穩定」。
2.  **直方圖**：確認製程是否「有能力」。
3.  **對比圖**：確認製程是否「均勻」。
