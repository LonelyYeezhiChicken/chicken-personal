# 👩‍💻 使用 DBeaver 連接

## 下載與預先準備

### DBeaver

[連結](https://dbeaver.io/download/)

### oracle11g

[安裝步驟](https://blog.lychicken.com/docs/daylily/postgreDaylily/setIndocker)

## 連線

### 1. 新增連線

- Ctrl + Shift + N
- Postgresql
- 下載相關 jar

> Host: localhost
>
> Port: 5432
>
> Database: postgres
>
> UserName: postgres

### 2. 測試連線

點選測試按鈕

畫面顯示連線成功即可

### 3. 連線

點選連線按鈕

即可連上資料庫
