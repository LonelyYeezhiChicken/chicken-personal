# 👩‍💻 MySQL 匯出資料庫腳本

## 環境

- DB GUI: MySQL Workbench 8.0 CE
- MySQL: 8
- OS: Windows11

## 步驟

1. 選單的 > Sever
2. Data Export
3. Tables to Export 勾選要匯出的資料庫
4. Export to Self-Contained File > 選擇匯出路徑
5. Include Create Schema
6. Start Export
