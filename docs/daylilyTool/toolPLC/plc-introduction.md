# 🏭 PLC 簡介

> **PLC (Programmable Logic Controller)**，中文稱為「可程式邏輯控制器」，是一種專為工業環境設計的數位電子操作控制器。它使用可程式記憶體來儲存指令，執行邏輯運算、順序控制、計時、計數和算術運算等功能，透過數位或類比的輸入/輸出模組，來控制各種機器或生產製程。

## PLC 的主要組成部分

一個典型的 PLC 系統主要由以下幾個部分組成：

1.  **中央處理單元 (CPU)**：
    -   PLC 的「大腦」，負責執行使用者程式、管理記憶體、處理輸入訊號、執行輸出操作等。
    -   通常包含微處理器和作業系統。

2.  **記憶體 (Memory)**：
    -   用於儲存使用者程式、輸入/輸出狀態、資料暫存器、計時器、計數器等。
    -   通常分為程式記憶體（儲存使用者邏輯）和資料記憶體（儲存變數和狀態）。

3.  **輸入/輸出模組 (Input/Output Modules)**：
    -   **輸入模組 (Input Module)**：將來自現場感測器（如開關、按鈕、光電感測器）的電氣訊號轉換成 CPU 能理解的數位訊號。
    -   **輸出模組 (Output Module)**：將 CPU 的控制訊號轉換成能驅動執行器（如馬達、電磁閥、指示燈）的電氣訊號。
    -   I/O 模組可以是數位式（開關量）或類比式（連續量）。

4.  **電源供應模組 (Power Supply Module)**：
    -   為 PLC 的內部電路和 I/O 模組提供穩定的工作電源。

5.  **程式設計設備 (Programming Device)**：
    -   通常是個人電腦，透過專用軟體和通訊介面（如 USB, Ethernet）連接到 PLC，用於編寫、下載、監控和除錯程式。

## PLC 在工業自動化中的作用

PLC 是現代工業自動化的基石，廣泛應用於各種製造業、流程控制和機械設備中，其主要作用包括：

-   **自動化控制**：取代傳統的繼電器控制電路，實現複雜的邏輯控制和順序操作。
-   **提高效率**：縮短生產週期，減少人工操作，提高生產效率和產品一致性。
-   **增強靈活性**：透過修改程式即可改變控制邏輯，無需重新接線，方便應對生產需求變化。
-   **提升可靠性**：採用工業級設計，抗干擾能力強，故障率低，能在惡劣環境下穩定運行。
-   **便於監控與診斷**：可以即時監控現場設備狀態，並透過程式進行故障診斷。

總之，PLC 讓工業設備變得「聰明」起來，能夠按照預設的邏輯自動完成各種任務，是實現智能製造不可或缺的技術。
